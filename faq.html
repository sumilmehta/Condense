FAQ

Frequently Asked Questions

These are some of the frequently asked questions

Which algorithm is being used to compress the files ?

We use the DEFLATE algorithm which is a combination of LZ77 and the Huffmann algorithm.

Will I be charged for using Condense ?

No. Condense is absolutely free to use.

Is the compression lossy or lossless ?

The compression being done is based on the DEFLATE Algorithm which is a lossless compression algorithm.

Will I be able to recover the original file from the compressed file ?

Presently we are not providing with the feature to recover the original file from the compressed file. However, we are planning to introduce this in the near future.

Is there any mobile application available for the same ?

Yes there is an app available at play store by the name CONDENSE.
