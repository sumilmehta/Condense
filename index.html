Key Features








Lossless Compression

Since, we are using the DEFLATE algorithm. The images we compress are lossless.

DEFLATE Algorithm

The algorithm we are using is a combination of LZ77 and Huffmann Algorithm. This is the same algorithm which is being used by 7zip.








Support for Mobile

Our website is fully responsive and is supported on various availabe devices and browsers.

Base64 Encryption

We also provide the feature of base64 encryption along with the string compression.
